## Installation

Clone the repository:

```bash
git clone https://github.com/Likhithpsdas17/likhith-portfolio.git
```

Navigate to project:

```bash
cd likhith-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
